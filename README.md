# NeuroBalance

NeuroBalance integrates a React frontend, AI notebooks, and blockchain modules for Aptos and Ethereum.

## Structure

- `frontend` React TypeScript app scaffold.
- `ai-model` Jupyter notebook with mock data.
- `blockchain/aptos` Move module.
- `blockchain/ethereum` Solidity contracts.
- `scripts/deploy` deployment placeholders.
- `docs` architecture overview.

## Setup

- Install Node.js and Yarn.
- Install Hardhat and Aptos CLI.

## Development

- Frontend dev server: `cd frontend && yarn install && yarn dev`.
